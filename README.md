# Ocean Quest

Ocean Quest is a Next.js starter for an ocean-industry-focused media and recruiting support service.

## Local Setup

```bash
npm install
npm run dev
```

## Content Integration Plan

- YouTube videos: fetch from YouTube Data API by playlist or channel.
- note articles: ingest RSS and enrich categories in a content table.
- eBooks: generate drafts from video transcripts, then publish reviewed PDFs.
- Events: keep an events table first, then connect to an external form or calendar later.
- Diagnosis: start with a local questionnaire and later send results to CRM or Sheets.
